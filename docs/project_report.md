# IoT Smart Agriculture Monitoring System

## Objective

To monitor agricultural conditions and automate irrigation decisions.

## Sensors

- Soil Moisture
- Temperature
- Humidity
- Light Intensity
- Water Level

## Technologies

- Python
- Streamlit
- ESP32
- Arduino
- Wokwi

## Features

- Real-time monitoring
- Alert generation
- Pump automation
- Dashboard visualization
- CSV logging
